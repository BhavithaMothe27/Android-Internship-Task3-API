# Task 3 - Backend Integration and API Development

Objective
Integrate backend services and REST APIs into an Android application.

Features
- Fetches user data from API
- Displays user names and emails
- Uses Retrofit for networking
- Parses JSON using Gson

Technologies Used
- Kotlin
- Jetpack Compose
- Retrofit
- Gson Converter

API Used
https://jsonplaceholder.typicode.com/users

Output
The application retrieves user data from the API and displays it in a scrollable list.
